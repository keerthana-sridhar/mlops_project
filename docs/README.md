# Malaria Classifier Documentation Pack

This folder contains the documentation and diagrams required for the assignment deliverables.

- `architecture.md`: System architecture with deployment and data flow diagrams.
- `high_level_design.md`: High-level design, components, and rationale.
- `low_level_design.md`: API definitions and I/O specifications.
- `test_plan.md`: Test strategy, test cases, and acceptance checks.
- `user_manual.md`: Non-technical user guide for operating the application.
- `acceptance_criteria.md`: Functional and operational acceptance criteria.
- `reproducibility.md`: What must be pushed, what stays local, and how to move the project to another laptop safely.
